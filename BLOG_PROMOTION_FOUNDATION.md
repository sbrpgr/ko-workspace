# 코워크스페이스 블로그 홍보 기초 자료

## 1. 서비스 공통 소개

코워크스페이스(ko-workspace)는 로그인 없이 브라우저에서 바로 사용할 수 있는 온라인 업무 도구 모음입니다. 텍스트 정리, AI 표 변환, CSV·엑셀 변환, 이미지 변환, QR 판독, EXIF 정리, PDF 편집, 자막 보정, 음성 받아쓰기, 녹음 파일 텍스트 변환, 녹음 파일 간편 편집, MD 파일 보기, 웹캠 녹화처럼 일상 업무에서 자주 필요한 작업을 한곳에서 처리할 수 있도록 구성했습니다.

서비스 주소: https://ko-workspace.com/

핵심 메시지:

- 설치 없이 웹브라우저에서 바로 실행
- 회원가입이나 로그인 없이 사용
- 텍스트, 표·CSV, 이미지, QR, PDF, 자막, 음성, 영상 작업 지원
- 대부분의 작업을 사용자 브라우저 안에서 처리
- 업무 중 짧게 반복되는 파일 변환, 정리, 복사 작업에 적합

공통 소개문 초안:

> 코워크스페이스는 텍스트, 표·CSV, 이미지, QR, PDF, 자막, 음성, 영상 관련 작업을 브라우저에서 바로 처리할 수 있는 무료 온라인 업무 도구 모음입니다. 별도 프로그램을 설치하거나 로그인하지 않아도 필요한 도구를 선택해 바로 사용할 수 있으며, 복잡한 편집 프로그램을 열기에는 부담스러운 작은 작업들을 빠르게 끝내는 데 초점을 맞췄습니다.

공통 CTA 문구:

> 지금 바로 코워크스페이스에서 필요한 도구를 열고 작업해 보세요.<br>
> https://ko-workspace.com/

블로그 글 공통 구성 권장안:

1. 이 기능이 필요한 상황
2. 코워크스페이스 도구로 할 수 있는 일
3. 사용 방법 3단계
4. 업무 활용 예시
5. 주의할 점 또는 팁
6. 서비스 링크 안내

공통 주의 문구:

> 코워크스페이스의 현재 도구들은 설치형 전문 프로그램을 완전히 대체하기보다, 자주 발생하는 간단한 정리와 변환 작업을 빠르게 처리하는 데 적합합니다.

## 2. 기능별 블로그 초안 자료

### 1. 음성으로 텍스트 쓰기

주소: https://ko-workspace.com/tools/voice-to-text/

추천 글 제목:

- 음성을 바로 글로 바꾸는 무료 받아쓰기 도구
- 회의 메모와 영상 대본 초안을 빠르게 만드는 방법

핵심 키워드:

- 음성 텍스트 변환
- 받아쓰기
- 회의록 초안
- 영상 대본 작성

상세 소개:

음성으로 텍스트 쓰기는 마이크로 말한 내용을 브라우저에서 실시간으로 텍스트로 받아 적는 도구입니다. 회의 중 떠오르는 내용을 빠르게 기록하거나, 영상 대본 초안을 말로 먼저 풀어내고 싶을 때 사용할 수 있습니다. 일반 메모뿐 아니라 유튜브 영상, 발표문, 회의 요약처럼 목적에 맞는 형태로 정리하는 데 활용할 수 있습니다.

사용 흐름:

1. 마이크 권한을 허용합니다.
2. 녹음 또는 받아쓰기를 시작합니다.
3. 말한 내용이 텍스트로 변환되는지 확인합니다.
4. 필요한 형식으로 정리한 뒤 복사하거나 저장합니다.

활용 예시:

- 회의 중 발언 내용을 빠르게 기록
- 강의나 발표 대본 초안 작성
- 유튜브 영상 스크립트 아이디어 정리
- 손으로 타이핑하기 어려울 때 말로 메모 작성

블로그 본문용 문단:

> 긴 글을 처음부터 타이핑하는 것이 부담스럽다면 먼저 말로 풀어낸 뒤 텍스트로 정리하는 방식이 효율적입니다. 코워크스페이스의 음성으로 텍스트 쓰기 도구는 브라우저에서 바로 실행되며, 마이크 권한만 허용하면 말한 내용을 실시간으로 텍스트화할 수 있습니다. 회의록, 강의 메모, 영상 대본처럼 초안이 필요한 작업에 잘 어울립니다.

### 2. 웹캠 녹화기

주소: https://ko-workspace.com/tools/webcam-recorder/

추천 글 제목:

- 설치 없이 웹캠 영상을 녹화하는 무료 브라우저 도구
- 발표 영상과 자기소개 영상을 빠르게 녹화하는 방법

핵심 키워드:

- 웹캠 녹화
- 브라우저 녹화
- 발표 영상 만들기
- 온라인 녹화 도구

상세 소개:

웹캠 녹화기는 카메라와 마이크를 이용해 브라우저에서 바로 영상을 녹화하는 도구입니다. 기본적인 웹캠 녹화뿐 아니라 좌우 반전, 필터, 밝기, 대비, 채도 조정, 배경 흐림 같은 간단한 화면 조정 기능을 제공합니다. 별도의 영상 편집 프로그램을 열기 전에 짧은 안내 영상이나 자기소개 영상을 빠르게 만들 때 유용합니다.

사용 흐름:

1. 카메라와 마이크 권한을 허용합니다.
2. 미리보기 화면에서 좌우 반전, 필터, 배경 효과를 조정합니다.
3. 녹화를 시작하고 필요한 내용을 말합니다.
4. 녹화를 종료한 뒤 결과 영상을 저장합니다.

활용 예시:

- 온라인 수업 안내 영상 제작
- 면접 연습 영상 녹화
- 자기소개 영상 제작
- 간단한 제품 설명 영상 녹화

블로그 본문용 문단:

> 짧은 안내 영상이나 발표 연습 영상을 만들기 위해 무거운 녹화 프로그램을 설치할 필요는 없습니다. 코워크스페이스 웹캠 녹화기는 브라우저에서 카메라와 마이크를 바로 사용해 녹화할 수 있으며, 화면 반전과 필터, 배경 흐림 같은 기본적인 조정도 함께 제공합니다.

### 3. AI 복붙 서식 정리

주소: https://ko-workspace.com/tools/ai-text-cleaner/

추천 글 제목:

- ChatGPT 답변의 별표와 마크다운을 깔끔하게 정리하는 방법
- AI 답변 복사 후 문서에 붙여넣기 좋은 형태로 바꾸기

핵심 키워드:

- AI 답변 정리
- ChatGPT 서식 제거
- 마크다운 정리
- 복붙 텍스트 정리

상세 소개:

AI 복붙 서식 정리는 ChatGPT, Claude, Gemini 등에서 복사한 답변의 별표, 제목 기호, 코드 블록, 링크 형식, 과한 줄바꿈을 문서에 붙여넣기 좋은 형태로 정리하는 도구입니다. AI 답변을 블로그 초안, 이메일, 보고서, 문서 작업에 활용할 때 불필요한 마크다운 표시를 줄이고 깔끔한 텍스트로 바꿔줍니다.

사용 흐름:

1. AI 답변이나 정리할 텍스트를 붙여넣습니다.
2. 제거하거나 유지할 서식 옵션을 선택합니다.
3. 정리 결과를 확인합니다.
4. 결과를 복사해 문서나 메일에 붙여넣습니다.

활용 예시:

- ChatGPT 답변을 보고서 문체로 정리
- 블로그 초안에서 마크다운 기호 제거
- 이메일에 넣기 전에 별표와 과한 줄바꿈 정리
- 메신저 공유용으로 긴 답변을 깔끔하게 변환

블로그 본문용 문단:

> AI 답변을 그대로 복사하면 `**강조**`, `### 제목`, 목록 기호 같은 마크다운 흔적이 남는 경우가 많습니다. 코워크스페이스의 AI 복붙 서식 정리 도구는 이런 표시를 빠르게 정리해 문서, 메일, 블로그 편집기에 바로 붙여넣기 좋은 텍스트로 바꿔줍니다.

### 4. 글자수 세기

주소: https://ko-workspace.com/tools/character-counter/

추천 글 제목:

- 공백 포함 글자수와 공백 제외 글자수를 한 번에 확인하는 도구
- 자기소개서, 블로그 원고, 광고 문구 글자수 확인하기

핵심 키워드:

- 글자수 세기
- 공백 제외 글자수
- 자기소개서 글자수
- 문장 길이 확인

상세 소개:

글자수 세기는 입력한 텍스트의 글자수, 공백 제외 글자수, 단어 수, 줄 수, 문단 수, 바이트 수 등을 빠르게 확인하는 도구입니다. 자기소개서, 과제, 블로그 원고, 광고 문구처럼 분량 제한이 있는 글을 작성할 때 유용합니다.

사용 흐름:

1. 확인할 텍스트를 입력합니다.
2. 공백 포함/제외 글자수를 확인합니다.
3. 단어 수, 줄 수, 바이트 수 등 추가 정보를 확인합니다.
4. 기준에 맞게 문장을 줄이거나 보완합니다.

활용 예시:

- 자기소개서 500자 제한 확인
- 블로그 메타 설명 길이 조정
- 광고 문구 분량 점검
- 과제 제출 전 글자수 확인

블로그 본문용 문단:

> 글자수 제한이 있는 글은 마지막에 직접 세기보다 작성 중간마다 확인하는 것이 좋습니다. 코워크스페이스 글자수 세기 도구는 공백 포함 글자수와 공백 제외 글자수를 함께 보여주기 때문에 제출 기준이 다른 글도 빠르게 점검할 수 있습니다.

### 5. 줄바꿈·공백 정리

주소: https://ko-workspace.com/tools/line-break-cleaner/

추천 글 제목:

- 복사한 글의 이상한 줄바꿈과 공백을 깔끔하게 정리하는 법
- PDF에서 복사한 텍스트를 문단 형태로 바꾸기

핵심 키워드:

- 줄바꿈 제거
- 공백 정리
- 문단 정리
- 복사 텍스트 정리

상세 소개:

줄바꿈·공백 정리는 PDF, 웹페이지, 메신저 등에서 복사한 텍스트의 어색한 줄바꿈과 과한 공백을 문단 형태로 정리하는 도구입니다. 문장 중간에서 끊어진 줄을 이어주거나, 여러 개의 빈 줄과 공백을 줄여 문서에 붙여넣기 좋은 상태로 바꿔줍니다.

사용 흐름:

1. 정리할 텍스트를 붙여넣습니다.
2. 줄바꿈 합치기, 빈 줄 정리, 공백 정리 옵션을 선택합니다.
3. 결과를 확인합니다.
4. 정리된 텍스트를 복사해 문서에 사용합니다.

활용 예시:

- PDF에서 복사한 본문 정리
- 메신저 대화 내용을 문단 형태로 정리
- 이메일에 넣기 전 줄바꿈 정리
- 블로그 초안의 과한 빈 줄 제거

블로그 본문용 문단:

> PDF나 웹페이지에서 글을 복사하면 문장 중간마다 줄이 끊기거나 공백이 어색하게 들어가는 일이 많습니다. 줄바꿈·공백 정리 도구를 사용하면 이런 텍스트를 문단 단위로 빠르게 정리해 문서 작업 시간을 줄일 수 있습니다.

### 6. 마크다운 편집기

주소: https://ko-workspace.com/tools/markdown-editor/

추천 글 제목:

- 설치 없이 쓰는 온라인 마크다운 편집기
- README와 블로그 초안을 마크다운으로 빠르게 작성하기

핵심 키워드:

- 마크다운 편집기
- Markdown 작성
- README 작성
- 온라인 문서 편집

상세 소개:

마크다운 편집기는 일반 텍스트를 제목, 목록, 인용, 코드, 링크 같은 마크다운 문법으로 편집하고 미리보기로 확인할 수 있는 도구입니다. 개발 문서, README, 블로그 초안, 업무 메모처럼 구조가 필요한 텍스트를 빠르게 정리할 때 적합합니다.

사용 흐름:

1. 문서 초안을 입력하거나 붙여넣습니다.
2. 제목, 목록, 링크 등 필요한 마크다운 문법을 적용합니다.
3. 미리보기로 결과 형태를 확인합니다.
4. 완성된 마크다운을 복사하거나 저장합니다.

활용 예시:

- GitHub README 초안 작성
- 블로그 글 구조 정리
- 회의 메모를 문서 형태로 정리
- 개발 문서나 안내문 작성

블로그 본문용 문단:

> 마크다운은 간단한 기호만으로 제목, 목록, 링크, 코드 블록을 표현할 수 있어 문서 초안 작성에 편리합니다. 코워크스페이스 마크다운 편집기는 브라우저에서 바로 작성하고 미리볼 수 있어 가벼운 문서 작업에 잘 맞습니다.

### 7. 이메일·URL·전화번호 추출기

주소: https://ko-workspace.com/tools/text-extractor/

추천 글 제목:

- 긴 텍스트에서 이메일, URL, 전화번호만 빠르게 추출하기
- 문의 내역과 문서에서 연락처 목록 정리하는 법

핵심 키워드:

- 이메일 추출
- URL 추출
- 전화번호 추출
- 연락처 정리

상세 소개:

이메일·URL·전화번호 추출기는 긴 텍스트 안에서 이메일 주소, 웹 링크, 전화번호만 골라 목록으로 정리하는 도구입니다. 문의 내역, 고객 메모, 공지문, 문서 본문처럼 정보가 섞여 있는 텍스트에서 필요한 연락처나 링크만 빠르게 분리할 수 있습니다.

사용 흐름:

1. 추출할 원문을 붙여넣습니다.
2. 이메일, URL, 전화번호 결과를 확인합니다.
3. 필요한 항목만 선택해 복사합니다.
4. 스프레드시트나 CRM 등에 붙여넣습니다.

활용 예시:

- 여러 XLSX 파일의 시트를 CSV로 일괄 변환
- 전화번호와 우편번호의 앞자리 0을 보존해 제출용 자료 만들기

블로그 본문용 문단:

> CSV 파일은 업무 시스템마다 인코딩과 구분자가 달라 엑셀에서 바로 열었을 때 글자가 깨지거나 숫자 앞자리 0이 사라질 수 있습니다. 코워크스페이스 CSV 엑셀 변환기는 CSV, TSV, XLSX를 브라우저에서 변환하면서 한국어 인코딩과 실무용 텍스트 보존 옵션을 함께 제공합니다.
