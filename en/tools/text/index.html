    Service Principles

    1. Open immediately Use tools without creating an account or signing in.
    2. Browser processing Whenever practical, text and files are processed in the browser instead of through an application server API.
    3. Privacy by default Input text, selected files, recordings, and generated results are not stored in a ko-workspace account database.
