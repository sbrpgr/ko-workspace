    Practical Guide

    영상과 맞지 않는 자막 싱크를 초 단위로 일괄 보정합니다

    자막 시간 보정은 SRT와 VTT 자막의 전체 시간을 앞이나 뒤로 이동하는 도구입니다. 영상 앞부분이 잘리거나 광고 구간이 빠지거나 인코딩 후 시작점이 달라지면 자막 전체가 일정하게 밀릴 수 있습니다.

    전체 자막이 늦게 나올 때

    자막이 영상보다 1.5초 늦게 보인다면 음수 값을 입력해 전체 시간을 앞당길 수 있습니다.

    오프닝 컷 편집 후

    영상 앞부분을 잘라낸 뒤 자막 시작점이 달라졌다면 컷 길이만큼 시간을 조정해 다시 맞춥니다.

    SRT와 VTT 모두 점검

    자막 형식을 자동으로 감지하거나 직접 선택해 시간 보정 결과를 확인하고 파일로 저장합니다.

    이 도구는 자막 한 줄씩 미세 조정하는 편집기가 아니라, 전체 싱크가 일정하게 밀린 경우 빠르게 보정하는 도구입니다. 일부 구간만 어긋난 자막은 전문 편집툴에서 세부 조정하는 것이 좋습니다.

    • 초 단위 오프셋을 적용해 전체 시간을 이동합니다.
    • SRT와 VTT 형식에 대응합니다.
    • 영상 편집 후 업로드 전 싱크 확인에 적합합니다.

    Service Principles

    서비스 원칙

    1. 즉시 실행 회원가입이나 로그인 없이 곧바로 도구를 열고 결과를 만들 수 있습니다.
    2. 브라우저 처리 도구는 가능한 한 서버 API 없이 브라우저 안에서 텍스트와 파일을 직접 처리합니다.
    3. 보안 우선 입력 텍스트·파일·녹화 데이터는 자체 서버에 저장하지 않고 100% 브라우저 안에서 처리합니다. 입력 데이터 저장 기능을 두지 않으며 회원 계정을 요구하지 않습니다.
    4. 광고 분리 작업 UI와 광고 영역을 분리해 도구 사용 흐름이 광고 때문에 흔들리지 않도록 구성합니다.
