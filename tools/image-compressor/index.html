    Practical Guide

    이미지 용량을 줄여 업로드와 공유를 가볍게 만듭니다

    이미지 용량 압축은 메일 첨부, 블로그 업로드, 쇼핑몰 상품 등록, 공공기관 제출처럼 파일 크기 제한이 있는 상황을 위해 만든 도구입니다. 품질과 최대 너비를 조정하며 결과 크기를 확인할 수 있습니다.

    첨부 제한에 맞추기

    메일이나 제출 시스템에서 파일이 너무 크다고 나올 때 품질과 너비를 조정해 제한 안에 들어가는 이미지를 만듭니다.

    웹 페이지 로딩 개선

    블로그나 상품 페이지에 들어갈 이미지를 적절히 줄이면 방문자가 더 빨리 이미지를 볼 수 있습니다.

    원본 보관과 압축본 사용

    원본은 그대로 두고 브라우저에서 만든 압축본만 저장해 제출용, 공유용으로 따로 관리할 수 있습니다.

    코워크스페이스는 이미지를 서버에 올려 압축 결과를 받는 방식 대신, 브라우저에서 사용자가 직접 품질과 크기를 확인하는 흐름을 택했습니다. 결과 품질은 원본과 설정에 따라 달라지므로 중요한 이미지는 압축 후 눈으로 확인하는 것이 좋습니다.

    • 품질과 최대 너비를 조정합니다.
    • 결과 파일 크기 변화를 확인합니다.
    • 메일, 블로그, 쇼핑몰, 제출용 이미지에 적합합니다.

    Service Principles

    서비스 원칙

    1. 즉시 실행 회원가입이나 로그인 없이 곧바로 도구를 열고 결과를 만들 수 있습니다.
    2. 브라우저 처리 도구는 가능한 한 서버 API 없이 브라우저 안에서 텍스트와 파일을 직접 처리합니다.
    3. 보안 우선 입력 텍스트·파일·녹화 데이터는 자체 서버에 저장하지 않고 100% 브라우저 안에서 처리합니다. 입력 데이터 저장 기능을 두지 않으며 회원 계정을 요구하지 않습니다.
    4. 광고 분리 작업 UI와 광고 영역을 분리해 도구 사용 흐름이 광고 때문에 흔들리지 않도록 구성합니다.
